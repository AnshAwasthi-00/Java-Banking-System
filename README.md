# Java Banking System

A console-based banking system built using Java with password-based authentication for secure transactions.

## Features
- Create multiple accounts
- Secure login with password authentication 🔐
- Deposit money
- Withdraw money
- Delete account
- Prevent duplicate account numbers
- Input validation
- Uses OOP concepts
- Uses ArrayList

## 🔐 Authentication System
- Each account is protected with a password
- Login is required before performing:
  - Deposit
  - Withdraw
  - Account deletion
- Ensures basic security for user accounts

## Concepts Used
- Encapsulation
- Object-Oriented Programming
- Collections (ArrayList)
- Switch-case
- Input validation
- Basic Authentication System
